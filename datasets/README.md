# Datasets

## CityLearn Challenge 2022

The `citylearn_datasets/` directory contains datasets from the [CityLearn Challenge 2022](https://www.aicrowd.com/challenges/neurips-2022-citylearn-challenge), redistributed from the [CityLearn](https://github.com/intelligent-environments-lab/CityLearn) package (MIT License).

### Contents

- **`citylearn_challenge_2022_phase_all/`** -- Full dataset with 17 residential single-family buildings, 8760 hourly timesteps (1 year). Used for Scenario S1 in the paper.
- **`citylearn_challenge_2022_phase_1/`** -- Phase 1 subset with 5 buildings. Used by the AmbitiousEngineers algorithm for pretraining.

## Demo Dataset

The `demo_1/` directory contains a lightweight synthetic dataset for quick testing and development. It is generated by the HEMS-Arena synthetic data pipeline and does not require external attribution.
